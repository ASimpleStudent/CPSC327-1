To enable C++11 support Click on project name->Properties-> C/C++ Build->Settings->GCC C++ Compiler-> Dialect ->'Language Standard' dropdown. ISO C++11(-std=c++0x) or later will be all you need in this class
To enable C++11 support for threads. If you are using threads you must add the pThread library-click on project name->Properties-> C/C++ Build->Settings->GCC C++ Linker-> Libraries -> add 'pthread' (with no '') to the Libraries (-l) list box on the right.
